## Edit MacOS Defaults

```bash
# Mojave anti-aliasing
defaults write com.microsoft.VSCode.helper CGFontRenderingFontSmoothingDisabled -bool NO
# Key Repeat
defaults write -g ApplePressAndHoldEnabled -bool false
```
